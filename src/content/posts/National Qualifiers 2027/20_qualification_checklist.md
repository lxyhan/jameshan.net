## Phase 0 -- Setup (This Week)

- [ ] Triathlon Ontario annual membership purchased ($84) -- valid to Mar 31, 2027
- [ ] Bookmark triathloncanada.com/calendar -- check for EOI links
- [ ] Create SPORTSTATS account at sportstats.ca -- for scouting competitors
- [ ] Set phone alarm for each EOI open date (see calendar above) -- Sun evening reminders

---

## Phase 1 -- ETA Aquathlon 4 (Mar 22 - Apr 10)

- [ ] **Mar 22 (Sun):** Check M20-24 EOI list on TC calendar -- scout field on SPORTSTATS
- [ ] **Mar 25 (Wed):** Decision: field looks weak -> GO. Fast competition -> SKIP to Phase 2
- [ ] **Apr 8 (Wed):** If GO: Register for race on ETA event website (Zone4/CCN)
- [ ] **Apr 10 (Fri 11\:30PM PST):** If GO: File EOI on Triathlon Canada calendar ($25) -- **HARD DEADLINE**
- [ ] **Apr 12:** Race day -- ETA Aquathlon 4 -- 1000m pool swim + 5000m indoor track (Edmonton)

---

## Phase 2 -- Beaumont + Gatineau (Jun 6 - Jul 4)

- [ ] **Jun 6 (Sat):** Check Beaumont M20-24 EOI list, scout field
- [ ] **Jun 24 (Wed):** If going: Register for Beaumont Triathlon race
- [ ] **Jun 26 (Fri 11\:30PM PST):** If going: File Beaumont EOI ($25) -- **HARD DEADLINE**
- [ ] **Jun 27:** Race day -- Beaumont Triathlon (Beaumont, AB)
- [ ] **Jun 13 (Fri):** Check Gatineau M20-24 EOI list, scout field
- [ ] **Jul 1 (Wed):** If going: Register for Triathlon de Gatineau
- [ ] **Jul 3 (Fri 11\:30PM PST):** If going: File Gatineau EOI ($25) -- Quebec license covers one-day fee -- **HARD DEADLINE**
- [ ] **Jul 4:** Race day -- Triathlon de Gatineau (Gatineau, QC)

---

## Phase 3 -- Do North + Lobster + Provincials (Jun 27 - Aug 27)

- [ ] **Jun 27 (Sat):** Check Do North M20-24 EOI list -- 3 spots, best odds in Canada
- [ ] **Jul 15 (Wed):** Register for Do North Events race (Zone4/CCN)
- [ ] **Jul 17 (Fri 11\:30PM PST):** FILE Do North EOI ($25) -- top priority race -- **HARD DEADLINE**
- [ ] **Jul 18-19:** Race day -- Do North Events Aquathlon -- Worlds course preview (Edmonton, AB)
- [ ] **Jul 26 (Sun):** Check Tri Lobster M20-24 EOI list -- 2 spots
- [ ] **Aug 12 (Wed):** If going: Register for Subaru Tri Lobster race
- [ ] **Aug 14 (Fri 11\:30PM PST):** If going: File Tri Lobster EOI ($25) -- **HARD DEADLINE**
- [ ] **Aug 16:** Race day -- Subaru Tri Lobster (Summerside, PE)
- [ ] **Aug 6 (Thu):** Check Provincials M20-24 EOI list -- only Ontario race
- [ ] **Aug 19 (Wed):** Register for Aquathlon Provincial Championships
- [ ] **Aug 21 (Fri 11\:30PM PST):** FILE Provincials EOI ($25) -- home race, zero travel -- **HARD DEADLINE**
- [ ] **Aug 27:** Race day -- Aquathlon Provincial Championships (Milton, ON)

## Phase 4 -- Cultus Lake (If Needed) (Aug 29 - Sep 20)

- [ ] **Sep 16 (Wed):** If going: Register for Cultus Lake Tri race
- [ ] **Sep 18 (Fri 11\:30PM PST):** If going: File Cultus Lake EOI ($25) -- **HARD DEADLINE**
- [ ] **Sep 20:** Race day -- Cultus Lake Tri (Chilliwack, BC)

---

## Phase 5 -- Post-Season Qualification Close (Oct Onwards)

- [ ] Monitor email from agegroupteams@triathloncanada.com -- direct qual: 4-8 weeks post-race
- [ ] Check TC website for roll-down -- must be top-8 + EOI filed -- first-come first-served
- [ ] If qualified: pay Team Management Fee (~$190) -- non-refundable, check email
- [ ] If qualified: order mandatory Team Canada uniform (~$350-400) -- race suit, jacket, hat, socks
- [ ] If qualified: register for 2027 Worlds race entry (Edmonton LOC) -- paid to Do North Events

## Eligibility

- ✅ Canadian Citizen
- ✅ Triathlon Ontario Annual Member (valid Mar 31, 2027)
- ☐ EOI filed per race ($25)
- ☐ Top-8 M20-24 finish (age-adjusted to Dec 31, 2027)
